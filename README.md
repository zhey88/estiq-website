# The Nail Artistry Website

A beautiful, responsive website for a nail salon business, inspired by The Nail Artistry Singapore.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and elegant design with smooth animations
- **Hero Section**: Eye-catching hero with announcement banner
- **About Section**: Tell your story and showcase your expertise
- **Services**: Highlight your nail services (Manicure/Pedicure, Nail Art, Nail Spa)
- **Gallery**: Showcase your best work with an interactive gallery
- **Multiple Locations**: Display all your business locations
- **Contact Integration**: WhatsApp and phone call integration
- **Social Media Links**: Connect to Instagram and Facebook
- **Smooth Scrolling**: Enhanced user experience with smooth navigation

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display & Montserrat)
- Unsplash Images (placeholder images)

## Customization Guide

### 1. Replace Images
Replace the Unsplash URLs with your own images:
- Hero background image
- About section image
- Service images
- Gallery images
- Location images

### 2. Update Content
- Edit business name, addresses, and contact information in `index.html`
- Update company registration details in the footer
- Modify service descriptions and offerings

### 3. Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #d4a574;    /* Main accent color */
    --secondary-color: #2c2c2c;  /* Dark color */
    --text-color: #333;          /* Text color */
    --light-bg: #f9f9f9;         /* Light background */
}
```

### 4. Add Your Logo
Replace the text logo in the navigation with an image:
```html
<div class="nav-brand">
    <img src="your-logo.png" alt="Your Business Logo">
</div>
```

### 5. Update Contact Information
- Phone numbers
- WhatsApp link
- Email address
- Physical addresses
- Social media links

## Setup Instructions

1. Download all files to a folder
2. Open `index.html` in a web browser to view locally
3. Customize the content, images, and colors
4. Deploy to a web hosting service (Netlify, Vercel, GitHub Pages, etc.)

## Deployment Options

### GitHub Pages (Free)
1. Create a GitHub repository
2. Upload all files
3. Enable GitHub Pages in repository settings

### Netlify (Free)
1. Sign up at netlify.com
2. Drag and drop your folder
3. Your site will be live instantly

### Vercel (Free)
1. Sign up at vercel.com
2. Import your project
3. Deploy with one click
